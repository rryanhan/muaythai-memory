# Training Method Badge System

## Purpose

Training Methods are core graph anchors and primary Library filters. They should eventually feel more memorable than text labels alone.

The first-pass direction is a badge-style SVG set inspired by Phosphor-style interface icons: bold, minimal, and easy to differentiate at small sizes.

## Badge Rules

- Format: SVG.
- Canvas: `64x64`.
- Outer shape: hexagon.
- Outer stroke: `3px`.
- Inner stroke: `3.2px`.
- Stroke caps and joins: round.
- No full bodies.
- Each context should be represented by one clear object or symbol.
- Avoid tiny illustration detail, dense handwraps, hatching, and anatomy-heavy linework.
- Motion, impact, reflection, or pressure marks stay abstract and controlled.

## Palette

Use the colors from `docs/visual-direction.md`:

- Graphite: `#171717`
- Border: `#D8D6CC`
- Red-orange: `#D14A32`
- Muted teal: `#256D85`
- Amber: `#8A6F3D`
- Surface: `#FFFFFF`

## First-Pass Badges

- Pad Work: a single Thai pad.
- Bag Work: a heavy bag.
- Partner Drill: two gloves or fists facing each other like a controlled clash.
- Clinch: linked hands, collar-tie grip, or interlocking grip symbol.
- Technical Work: a motion path, stance marks, or a simple trajectory/axis symbol.

Shadowboxing is no longer part of the Muay Thai training method badge set. It should be a standard Tag or practice format under another context, usually Technical Work.

Warmup is no longer part of the Muay Thai training method badge set. It should be reserved for the later workout or conditioning system.

## Workflow

Generate the full set together first to evaluate consistency. Refine individual badges one by one after seeing the set in an icon board.

Generated files live in `assets/context-badges/`.
