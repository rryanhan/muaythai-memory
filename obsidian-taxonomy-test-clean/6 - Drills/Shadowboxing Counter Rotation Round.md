# Shadowboxing Counter Rotation Round

## Summary
A shadowboxing round built around loading counters through rotation instead of throwing arm-only shots.

## Steps
1. Start each exchange by imagining an incoming straight punch.
2. Slip or shoulder turn to load the counter side.
3. Throw uppercut or hook from the loaded rotation.
4. Move the feet so the counter does not leave you square.
5. Finish each sequence with an exit step.

## Notes
This directly supports the slip-right step-through uppercut drill.
