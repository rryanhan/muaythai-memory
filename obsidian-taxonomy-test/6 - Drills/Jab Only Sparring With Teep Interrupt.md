# Jab Only Sparring With Teep Interrupt

## Summary
Constraint sparring where both partners use only jabs, with the defender allowed to interrupt forward pressure using the lead teep.

## Steps
1. Both partners begin at light contact.
2. Only jabs are allowed for offense.
3. If one partner pressures too directly, the other may use a lead teep to the body.
4. After each teep, both reset distance and continue.
5. Focus on timing and distance rather than winning the exchange.

## Notes
This makes the teep feel like a range tool, not just a strike.
